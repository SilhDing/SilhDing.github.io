## 异常处理的标准模型
```java
class MyMath {
	public static int div(int a, int b) throws Exception {
		int result = 0;
		System.out.println("BEGIN!");
		try {
			result = a/b;
		} catch (Exception e) {
			throw e;
		} finally {	  
		System.out.println("END!");
		}
		return result;
	}
}

// 请留意这个模型，
// 上面的catch语句可以删去，然后产生的异常会顺着throws到调用处。

public class 异常处理 {
	public static void main(String args[]) {

		try {
			System.out.println(MyMath.div(10,0));   //实例化
		} catch (Exception e) {
			e.printStackTrace();
		} 
		

	}
}
```
