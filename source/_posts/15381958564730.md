## 一个谷歌电面题

求一个树的所有叶子节点，但是用iterative的方法

```java
public class test1 {

	public List<Integer> getLeaves(Node root) {

		List<Integer> res = new ArrayList<>();

		Node node = pushLeft(root);

		while (node != null) {
			if (node.left == null && node.right == null) {
				res.add(node.val);
				
				if (node.parent == null) return res;
				else if (node == node.parent.left) {  // 当前节点是其parent的左节点
				    // 如果右边还有东西
				    if (node.parent.right != null) node = pushLeft(node.parent.right);
				    // 右边没有东西，则跳到parent的parent
				    else node = node.parent.parent;
				} else  {    
					// 当前节点是其parent的右节点
					// 那么以这个节点为最右边节点的树完成遍历
					while (node.parent != null && node == node.parent.right) {
						node = node.parent;
					}
					node = node.parent;
				} 

			} else if (node.right != null) {
				node = pushLeft(node.right);
			} 
		}
		return res;
	}

	private Node pushLeft(Node node) {
		if (node == null) return null;
		while (node.left != null) {
			node = node.left;
		}
		return node;
	}
}
```

这个问题的关键在于得到一个叶子之后，怎么往上走。注意这里提供的parent node。但是没有parent node的话，又该怎么办？
